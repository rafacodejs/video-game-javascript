## 📜 Resumen 📜

Juega y diviértete con SuperBomb Boy. Juego retro construido de manera pura con código legacy en HTML, CSS Y JavaScript, utilizando canvas y adaptado a dispositivos móviles, junto a funciones para adaptar su medidas y elementos al viewport de cada pantalla.

## 💻 Si quieres ver el código en tu computadora solo debes hacer esta instalación 💻

- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/rafacodejs/video-game-javascript.git
  - cd video-game-javascript
  - Abrir tu editor de código
